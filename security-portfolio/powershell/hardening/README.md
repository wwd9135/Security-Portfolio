# Hardening

Endpoint configuration and compliance scripts deployed via Microsoft Intune.

## Folders

| Folder | Contents |
|---|---|
| [`dell-command-update/`](dell-command-update/) | PSADT v4 Win32 apps for BIOS and driver updates via Dell Command Update |
| [`intune-remediations/`](intune-remediations/) | Proactive Remediation detection + remediation script pairs |
| [`bitlocker-management/`](bitlocker-management/) | PSADT v4 Win32 app for TPM+PIN BitLocker setup (MBAM replacement) |

See each subfolder's README for architecture details, prerequisites, and exit codes.
