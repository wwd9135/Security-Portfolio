# PowerShell

Operational PowerShell scripts organised by function.

## Folders

| Folder | Contents |
|---|---|
| [`automation/`](automation/) | Graph API policy search, connectivity testing |
| [`hardening/`](hardening/) | Endpoint configuration: DCU, Intune Proactive Remediations, BitLocker |
| [`incident-response/`](incident-response/) | ASR audit-mode setup, AD password audit workflow |
| [`reporting/`](reporting/) | Intune compliance export, Windows Update diagnostics, GPO auditing |
| [`modules/`](modules/) | Shared PSADT configuration module |

Each subfolder contains its own README with prerequisites and usage examples.
